디바이스 색상

색 하나가 보고마다 다르게 불려서, 세 가지 이름을 묶어 둡니다. 실시간에서 팔면 물자에서 그 색이 빠지고 마감에 반영돼요.

실시간물자마감랭킹
예) 실시간 블루 = 물자 네이비 = 마감 블루 — 같은 줄에 적으면 하나의 디바이스로 계산됩니다.

세일즈 랭킹 연동

켜두면 실시간 「판매 기록 +」를 누를 때 PLOOM 세일즈 랭킹 앱에도 판매 대수가 자동 등록됩니다. (카운트만)

랭킹 앱에 없는 이름이면 새로 등록됩니다. 이미 있으면 PIN이 맞아야 연결돼요.
랭킹 앱에서 직접도 등록하고 있다면 두 번 세어질 수 있어요. 둘 중 하나만 쓰세요.

점포 재고 색상

오전/오후 점포 재고에서 탭할 색상 목록입니다. (판매 집계와는 무관 · 표시용)

구매혜택 종류

실시간 세일즈에서 입력할 구매혜택 목록입니다. 이름에 ‘스타벅스’가 들어간 항목은 물자/마감의 스타벅스 카드 소진과 연결됩니다.

판매 기록 초기화

실시간 판매 누적 기록과 물자 기록을 모두 지웁니다. 일정은 남습니다.

월간 일정 — 엑셀 가져오기

매달 받는 엑셀(.xlsx)을 그대로 선택하세요. 팀·날짜·점포·주소·점포코드를 알아서 읽어 날짜별로 채웁니다.

헤더는 팀 / 세일즈 진행 일차 / 날짜 / 요일 / 점포/흡연구역 / 점포 타입 / 점포명 / 주소 / 점포코드 를 인식합니다.
